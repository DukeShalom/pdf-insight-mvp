# 🌟 Solace — AI-Powered Emotional Journal with Voice

A beautiful, compassionate web app for daily emotional journaling with AI responses, mood tracking, and **full voice interaction support**.

---

## ✨ What's New: Voice Features

Users can now **speak their feelings** and **hear Solace's responses** read aloud.

### 🎤 Speech-to-Text
- Tap "🎤 Voice" to record journal entries
- Live transcription appears as you speak
- Animated waveform shows recording is active
- Auto-detects end of thoughts
- Edit transcription before submitting

### 🔊 Text-to-Speech
- Tap "🔊 Play Response" to hear Solace's voice
- Warm, natural voice synthesis
- Easy play/stop controls
- Works perfectly on mobile

### ✏️ Choose Your Mode
- **Type** — Traditional journaling
- **Voice** — Hands-free, natural speaking

## 🚀 How to Use

### 1. Start App
Open `index.html` in a modern browser

### 2. Complete Onboarding
- Select your reason for journaling
- Set initial mood (1-5 slider)
- Enter your name

### 3. Journal via Text or Voice
**Text Mode:**
- Type thoughts in textarea
- Submit with "Send to Solace ♥"

**Voice Mode:**
- Tap "🎤 Voice" button
- Tap "🎤 Start Recording"
- Speak naturally (watch waveform)
- Tap "⏹️ Stop" when done
- Review transcription
- Submit entry

### 4. Receive AI Response
- Read or listen to Solace's response
- See follow-up prompts
- Click prompts to continue conversation (future feature)

### 5. Track Mood
- View 7-day history in "History"
- See mood chart in "Mood" page
- Read weekly insights
